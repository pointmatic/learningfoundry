# learningfoundry

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

A curriculum engine that turns a YAML curriculum definition into a deployable SvelteKit learning application — with LLM-generated content, interactive assessments, executable notebooks, and data visualizations — in a single pipeline.

## Installation

```bash
pip install learningfoundry
```

## Usage

```bash
learningfoundry build
```
